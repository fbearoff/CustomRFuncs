# CustomRFuncs

My custom R functions

## returnPCA
Based on the DESeq2 function `plotPCA`, returns PCA data for the first 4 principal components.

## plotGene
Returns a boxplot of a gene_symbol from a DESeq2 dds object by cross-referencing a tx2gene object.

## plotMA
Returns a MA plot of a DESeq2 results function for the queried conditions.

## compDESeq2
Returns a `data.table` object from a DESeq2 `results` function with gene names and other relevant metadata
